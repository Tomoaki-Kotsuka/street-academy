# Streetacademy Courses

ストアカ で公開している講座の教材を、このリポジトリにまとめています。

## 講座一覧

| 講座 | フォルダ | 概要 |
|---|---|---|
| 事務職のための Python 入門 | [`python-data-basics/`](./python-data-basics/) | Excel・VBA は使えるけど Python は初めての方向け。pandas で Excel 自動化、matplotlib で可視化、最終回に「月次レポート自動生成」プロジェクトを作る。 |

今後、別の講座を作った場合はここに追加していきます。

## このリポジトリの使い方

各講座の教材は **Google Colab で実行** することを前提としています。
受講者の方は、各講座フォルダ内の `README.md`（例: [`python-data-basics/README.md`](./python-data-basics/README.md)）にあるセットアップ手順に従って進めてください。
